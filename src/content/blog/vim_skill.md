# Vim 批量在行首加 # 注释（多种场景）
## 1. 手动选中区间（可视化模式，最常用）
1. 打开文件，移动光标到**起始行**
2. 按 `V` 进入**行可视化选中**
3. 上下移动光标选中需要注释的多行
4. 输入 `:` ，底部出现 `:'<,'>`
5. 输入命令：`s/^/#/`
```
:'<,'>s/^/#/
```
回车，选中所有行开头加上 `#`

## 2. 指定行号范围（精准，不用手动选）
格式：`起始行,结束行s/^/#/`
### 示例
给 10～20 行全部注释：
```vim
:10,20s/^/#/
```
给第5行到文件末尾注释：
```vim
:5,$s/^/#/
```
给当前行往下10行注释（当前行是 `.`）：
```vim
:.,.+10s/^/#/
```

### 命令拆解
- `s/^/#/`
  - `^`：匹配**行开头**
  - `#`：在行首插入 #

## 3. 取消注释（去掉行首#）
选中范围后执行：
```vim
:'<,'>s/^#//
```
行号示例（10-20取消注释）：
```vim
:10,20s/^#//
```

## 4. 进阶：行首有空格，# 加在空格后
如果代码前面有缩进空格，不想把#塞最前面：
```vim
:'<,'>s/\(\s*\)/\1#/
```
效果：
```
    test
→
    #test
```

## 5. 一键重复执行（不用重新输命令）
注释完一批，还要注释另一块，直接按 `@:` 重复上一条替换命令。
